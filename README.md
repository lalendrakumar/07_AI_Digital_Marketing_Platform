## Project Version 1.0 – Marketing Campaign Analyzer

### Objective
Analyze digital marketing campaign performance using Python.

### Tools Used
- Python
- Pandas
- NumPy
- VS Code
- Jupyter Notebook

### Key Metrics
- CTR
- Conversion Rate
- Cost per Conversion
- ROI

### Python Concepts Applied
- Basic Data Types (`int`, `float`, `str`, `bool`)
- Type checking and conversion
- Boolean logic for business decisions

### Python Data Types Applied
- Integers (`int`) → impressions, clicks, conversions
- Floating point (`float`) → cost, revenue, ROI
- Strings (`str`) → campaign_id, channel
- Boolean (`bool`) → is_profitable
- Typecasting used to avoid TypeErrors
- String methods applied for data cleaning


### Outcome
Identified top-performing marketing channels based on ROI and conversion efficiency.
